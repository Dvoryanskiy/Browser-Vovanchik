# Мой Браузер на основе Chromium

Браузер, построенный на базе Electron (Chromium).

## Возможности

- ✅ Множественные вкладки
- ✅ Навигация (назад/вперед/обновить)
- ✅ Адресная строка с поддержкой поиска
- ✅ Современный интерфейс
- ✅ Горячие клавиши

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

Для запуска в обычном режиме:
```bash
npm start
```

Для запуска в режиме разработки (с открытыми DevTools):
```bash
npm run dev
```

## Горячие клавиши

- `Ctrl+T` / `Cmd+T` - Новая вкладка
- `Ctrl+W` / `Cmd+W` - Закрыть вкладку
- `Alt+←` - Назад
- `Alt+→` - Вперед
- `Ctrl+R` / `Cmd+R` - Обновить
- `F11` - Полный экран

## Структура проекта

- `main.js` - Главный процесс Electron
- `preload.js` - Скрипт предзагрузки для безопасной связи между процессами
- `index.html` - Интерфейс браузера
- `styles.css` - Стили интерфейса
- `renderer.js` - Логика работы браузера (вкладки, навигация)
